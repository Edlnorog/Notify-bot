Помощь людям с ограниченными возможностями. Создание сайта для экстренной помощи

Вступление

В современном мире доступная среда и равные возможности для всех граждан — не просто норма цивилизованного общества, а насущная необходимость. Однако люди с ограниченными возможностями здоровья (ОВЗ) и маломобильные группы населения (пожилые, временно травмированные) до сих пор сталкиваются с серьезными трудностями при передвижении по городу. Отсутствие специализированного транспорта, сложности с вызовом такси и ограниченная доступность общественных мест создают барьеры для их самостоятельности и социальной активности.

Цель:

Создание сайта для взаимодействия волонтеров и лиц с ОВЗ в г. Москва.

Задачи:

  • создать дизайн главных страниц;
  • разработать интерактивные элементы для главной страницы;
  • верстка сайта;
  • создать базы данных;
  • создать политику конфиденциальности;
  • адаптировать сайт под мобильные устройства;
  • запустить продвижение сайта в соцсетях;
  • составить диаграмму Ганта;
  • составить проектную техническую документацию;
  • подготовить пояснительную записку;
  • подготовить презентацию к защите проекта;
  • защитить проект.
